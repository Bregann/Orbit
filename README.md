# FinanceManager

A C#/.NET/React application for managing personal finances: track transactions, categorize spending and manage budgets and accounts


## Features

- Accounts: Access via Open Banking API/Monzo API
- Transactions: Add in via APIs, manage and add to pots
- Budgets: monthly budget based on income, category limits, rollovers

## Tech Stack
- Language: C#, React
- Runtime: .NET 9
- UI: React
- Data: SQLite / PostgreSQL
- Testing: NUnit
